# E-Commerce Dashboard Project

## Project Overview
This project analyzes e-commerce sales data from Olist to uncover business insights related to revenue, customer behavior, product performance, and operational efficiency.

Using SQL, Python, and dashboarding tools, this project identifies key trends and provides actionable recommendations to improve business performance.

---

## Business Problem
The business aims to understand:
- Revenue trends over time
- Product category performance
- Customer satisfaction drivers
- Seller performance
- Delivery efficiency

---

## Objectives
- Analyze revenue and order trends
- Track business KPIs
- Identify top-performing categories and sellers
- Measure customer satisfaction
- Build an executive dashboard

---

## Tech Stack
- SQL
- Python
- Pandas
- Matplotlib
- Power BI / Tableau
- GitHub

---

## KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Customer Satisfaction Score
- Delivery Time
- Top Categories
- Seller Performance

---

## Workflow
1. Data Collection
2. Data Cleaning
3. SQL Analysis
4. Exploratory Data Analysis
5. Dashboard Development
6. Business Recommendations
